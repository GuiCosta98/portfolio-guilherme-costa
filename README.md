# Portfólio Pessoal - Guilherme Costa

## Sobre o Projeto

Este portfólio pessoal é uma aplicação web estática desenvolvida com HTML e CSS puro. O projeto foi desenvolvido como um exercício prático para consolidar conhecimentos em HTML5, CSS3 e design responsivo, com foco em criar uma interface moderna, acessível e visualmente atrativa.

## Tecnologias Utilizadas

- Linguagens: HTML5, CSS3
- Layout: CSS Grid e Flexbox
- Fontes: Google Fonts (Sen e Inter)
- Icones: SVG
- Ferramentas: Git, GitHub, VS Code
- Hospedagem: GitHub Pages 

## Como Executar

1. Clone o repositório: git clone https://github.com/GuiCosta98/portfolio.git
3. Acesse a pasta do projeto: cd portfolio
3. Abra diretamente o arquivo index.html em qualquer navegador moderno
4. Para visualização online, o projeto está disponível em: https://guicosta98.github.io/portfolio/


## Funcionalidades

### Implementadas

- Design responsivo com Grid e Flexbox
- Animações CSS (fade-in na entrada e hover effects nos ícones)
- Layout estruturado em seções claramente definidas
- Formulário de contato (interface)
- Links sociais integrados (LinkedIn, GitHub, CV)
- Tema escuro com paleta de cores consistente
